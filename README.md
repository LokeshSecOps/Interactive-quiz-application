# 🎯 Interactive Quiz Application

This is a responsive and interactive quiz app built using **HTML, CSS, and JavaScript**. It features 15 multiple-choice questions, instant feedback, score tracking, and a clean user interface. Developed as part of the **CODTECH Internship Task-1**.

---

## 🚀 Features

- ✅ 15 curated quiz questions
- ✅ Instant feedback on answers
- ✅ Score tracking and final result display
- ✅ Dynamic question loading
- ✅ Responsive design for desktop and mobile
